从零开始了解人工智能的核心概念
🔍
术语/名词 专业解释 小学生大白话
🎯 基础概念
AI (人工智能) Artificial Intelligence，指让计算机模拟人类智能行为的技术，包括学习、推理、感知等能力。 就像给电脑装了一个"大脑"，让它能像人一样思考和学习。
ML (机器学习) Machine Learning，是AI的一个分支，通过算法让计算机从数据中自动学习规律，无需明确编程。 就像教小孩认字，给电脑看很多例子，它就能自己学会规律。
DL (深度学习) Deep Learning，使用多层神经网络来模拟人脑神经元工作方式的机器学习方法。 模仿人脑的结构，用很多层"神经元"来处理信息，就像大脑有很多层一样。
神经网络 Neural Network，模拟生物神经系统的数学模型，由相互连接的节点（神经元）组成。 像人脑里的神经元一样，很多小"开关"连在一起，互相传递信息。
🧠 大模型相关
LLM Large Language Model，大型语言模型，是用海量文本数据训练的深度学习模型，能理解和生成人类语言。 就像一个读了很多很多书的超级聪明学生，能理解你说的话，也能写出很棒的文章。
GPT Generative Pre-trained Transformer，生成式预训练变换器，是一种基于Transformer架构的语言模型。 像一个很会写作文的机器人，先学了很多知识，然后能写出各种文章。
Transformer 一种神经网络架构，使用"注意力机制"来处理序列数据，是现代大语言模型的基础。 像一个很专注的学生，能同时注意到句子里每个词的重要性。
预训练 Pre-training，在大量无标签数据上训练模型，让模型学习语言的基本规律和知识。 就像让孩子先大量阅读各种书籍，打好基础知识。
微调 Fine-tuning，在预训练模型基础上，用特定任务的数据进行进一步训练。 就像已经会写作文的学生，再专门练习写诗歌或小说。
🤖 Agent相关
Agent 智能代理，能够感知环境、制定计划、执行行动并完成特定任务的AI系统。 像一个能干的助手，能理解你的需求，制定计划，然后帮你完成任务。
Multi-Agent 多智能体系统，多个Agent协同工作，通过交流合作完成复杂任务。 就像一个团队里有很多专家，每个人负责不同的事情，大家一起合作。
Tool Use 工具使用，Agent能够调用外部工具或API来扩展自己的能力。 就像人会使用计算器、查字典一样，AI也能使用各种工具来帮助解决问题。
Planning 规划能力，Agent能够分解复杂任务，制定步骤清晰的执行计划。 就像做作业前先列个计划，先做什么后做什么，一步一步来。
Memory 记忆系统，Agent能够存储和回忆之前的交互信息和学到的知识。 就像人的记忆一样，能记住之前聊过的内容和学过的东西。
⚙️ 技术参数
Token 词元，是模型处理文本的基本单位，可以是一个字、词或字符。 就像把句子切成小块，每一小块就是一个token，电脑就是这样理解文字的。
Context Length 上下文长度，模型一次能处理的最大token数量。 就像人的短期记忆，一次能记住多少内容，超过了就会忘记前面的。
Temperature 温度参数，控制模型输出的随机性，数值越高输出越有创意但可能不准确。 就像调节创造力的旋钮，调高了回答更有想象力，调低了回答更稳定准确。
Top-k 在生成下一个词时，只考虑概率最高的k个候选词。 就像选择题只给你前几个最可能的答案选项。
Top-p 核采样，选择累积概率达到p的最小候选词集合。 就像说"我要选概率加起来达到90%的那些选项"。
📚 训练相关
Dataset 数据集，用于训练模型的大量数据集合。 就像学生的课本和练习册，模型就是从这些"教材"里学知识的。
Epoch 训练轮次，模型完整学习一遍所有训练数据称为一个epoch。 就像学生把所有课本完整复习一遍，叫做一轮复习。
Batch Size 批次大小，每次训练时同时处理的数据样本数量。 就像老师每次批改多少份作业，一次改得多就快但累，改得少就慢但轻松。
Learning Rate 学习率，控制模型参数更新速度的超参数。 就像学习的速度，太快可能学不扎实，太慢又效率低下。
Overfitting 过拟合，模型在训练数据上表现很好，但在新数据上表现差。 就像死记硬背课本，考试题一模一样就会，题目稍微变化就不会了。
🔌 应用接口
API Application Programming Interface，应用程序编程接口，允许不同软件之间通信的规范。 就像餐厅的菜单，告诉你可以点什么菜，怎么点菜。
SDK Software Development Kit，软件开发工具包，包含开发应用所需的工具和文档。 就像一个工具箱，里面有各种工具和说明书，帮你造房子。
REST API 基于HTTP协议的API设计风格，使用标准的HTTP方法进行数据交互。 就像标准化的邮寄系统，大家都按同样的方式寄信收信。
Endpoint API的访问点，是一个特定的URL地址，用于接收和处理请求。 就像每个商店的门牌号，你知道地址就能找到这家店。
💬 提示工程
Prompt 提示词，输入给模型的指令或问题，用来引导模型生成期望的输出。 就像跟聪明的朋友说话，你说得越清楚，他就能更好地帮助你。
Few-shot 少样本学习，在提示中给出几个示例，让模型学习任务模式。 就像给孩子看几个例子："看，这样做是对的"，然后让他照着做。
Zero-shot 零样本学习，不给任何示例，直接让模型完成任务。 就像直接告诉一个聪明的学生"帮我写首诗"，不给任何例子。
Chain-of-Thought 思维链，引导模型逐步推理，展示思考过程。 就像解数学题时要求"写出解题过程"，让模型一步步思考。
📊 评估指标
Accuracy 准确率，模型预测正确的样本数占总样本数的比例。 就像考试成绩，100道题对了80道，准确率就是80%。
Precision 精确率，模型预测为正例中真正是正例的比例。 就像说"这是苹果"的时候，真的是苹果的比例有多高。
Recall 召回率，所有真正的正例中被模型找出来的比例。 就像一堆水果里的所有苹果中，你能找出多少个。
F1 Score F1分数，精确率和召回率的调和平均数，综合评估模型性能。 就像期中期末成绩的平均分，综合考虑两个方面的表现。
🚀 新兴概念
RAG Retrieval-Augmented Generation，检索增强生成，结合信息检索和文本生成的技术。 就像考试时可以查资料，先找到相关信息，再根据信息回答问题。
Vector Database 向量数据库，专门存储和检索高维向量数据的数据库系统。 就像一个特殊的图书馆，能根据内容的"相似度"快速找到你要的书。
Embedding 嵌入向量，将文本转换为数字向量的过程，相似内容的向量会很接近。 就像给每个词语一个"身份证数字"，意思相近的词数字也很相近。
RLHF Reinforcement Learning from Human Feedback，基于人类反馈的强化学习。 就像训练宠物，做得好就夸奖，做得不好就纠正，让它越来越听话。
Hallucination 幻觉，AI模型生成看似合理但实际错误或无根据的信息。 就像做梦时觉得很真实，但醒来发现是假的，AI有时也会"说梦话"。
